# 产品定位

## 产品定位

本产品是一个“企业决策前的仿真顾客 Agent 工作台”。

更完整的定位是：

面向企业决策者，通过平台型 Agent 工作台调度 Skill、Tool、资源库、知识库和仿真顾客群体，帮助企业在关键决策正式执行前，模拟顾客反应、识别认知盲区、发现转化阻力，并生成可执行的决策建议。

## 核心口号

候选口号：

1. 让每个重要决策，先经过顾客视角验证。
2. 用仿真顾客预演企业决策。
3. 在投入资源前，先看顾客会怎么反应。
4. 把顾客视角、企业资源和业务知识带进每一次决策。
5. 面向企业决策的 Agent 工作台。

## 用户心智

用户打开产品时，心里的问题不是：

“帮我随便聊一聊。”

而是：

“我今天要验证什么决策？”
“这个决策需要调用哪些数据、知识和工具？”
“哪些顾客会支持、反对或犹豫？”
“哪里有风险？”
“下一步应该直接推进、修改方案，还是先测试？”

## 首页心智

首页不是传统后台首页，也不是最终决策报告页。

首页应该像平台型 Agent 工作台：

* 左侧展示工作区、项目、Skill、Tool、资源库、知识库、历史记录和我的。
* 右侧提供直接任务输入、快捷功能和业务模板。
* 中央输入框是“任务启动器”，不是闲聊窗口。
* 模板帮助用户快速进入定价、新产品、营销文案、品牌定位、会员权益等业务场景。

## 产品与传统工具的区别

传统问卷工具解决：如何收集用户反馈。

传统 BI 工具解决：过去发生了什么。

传统咨询报告解决：如何分析市场和竞争。

通用 Chatbot 解决：临时问答和内容生成。

本产品解决：在决策发生之前，调用企业资源、知识和仿真顾客，推演目标顾客可能如何理解、反应、犹豫、反对和行动。

## 产品不应该做成什么

不要做成：

* 通用聊天机器人
* 只会回答问题的 AI 助手
* 只生成用户画像的工具
* 只生成调研问卷的工具
* 只做行业报告的工具
* 只分析历史数据的 BI 系统
* 只服务服装行业或爆款分析的产品
* 一打开就是复杂表单的后台系统

## 产品应该做成什么

应该做成：

* 企业决策 Agent 工作台
* 顾客反应模拟器
* 决策风险雷达
* 方案优化工作台
* 多角色顾客推演平台
* 企业资源与知识调度平台
* 决策报告生成系统
* 决策复盘系统

## 产品边界

本产品不承诺完全预测真实市场结果。

本产品的目标是：

* 帮助用户提前发现可能的顾客反应
* 帮助用户识别方案盲区
* 帮助用户调用企业资源和知识辅助决策
* 帮助用户比较不同方案
* 帮助用户形成更清晰的下一步行动
* 帮助用户降低决策试错成本

## 核心差异化

本产品的差异化不在于“生成内容”，而在于：

1. 用平台型 Agent 工作台组织任务入口
2. 围绕企业决策组织流程
3. 调度 Skill、Tool、资源库和知识库
4. 使用多类仿真顾客进行反馈
5. 输出支持、反对、犹豫和条件接受
6. 识别风险与争议
7. 给出方案修改建议
8. 沉淀项目、历史记录和复盘

## 竞品与 UI 参考

本项目可以参考多个平台型 Agent 产品的界面范式，但只参考 UI 布局和交互组织，不复制其产品主线。

### AnyGen

参考重点：

* 平台型 Agent 首页结构
* 左侧工作区导航
* 中间任务启动器
* 快捷能力入口
* 模板区

不参考：

* 泛内容生成工具定位
* 过多通用创作模板

### ChatGPT

参考重点：

* 左侧最近记录
* 新聊天 / 搜索 / 文件库 / 项目等基础导航组织
* 简洁、低干扰的侧栏视觉

不参考：

* 通用聊天机器人产品心智
* 以纯对话消息流承载全部任务

### Lovart

参考重点：

* 大输入框上下的极简启动结构
* 胶囊式能力按钮
* 低噪音的中心聚焦布局

不参考：

* 设计工具主线
* 以视觉创作为核心的任务结构

### Converge AI

参考链接：

https://enter.converge.ai/workspace/8fe43b52-efb8-40b8-83a5-59aa0d20560d?tab=newchat

当前备注：

该链接可能需要登录态或 workspace 权限，公开环境未必能直接访问。后续 UI 调整时，应以用户提供的截图或可访问页面为准。

参考重点：

* workspace 级页面布局
* 新建对话 / 新建任务入口
* 左侧工作区与历史记录的空间组织
* 中间任务输入区的层级和留白
* 多入口但低噪音的 Agent 工作台感

不参考：

* 若其产品主线偏通用问答或通用 Agent，本项目不跟随。
* 不取消“仿真顾客在场”的右侧顾客心声 / 顾客留声设计。
* 不把任务详情页退化成普通聊天页。

本产品最终 UI 应综合参考 AnyGen、ChatGPT、Lovart 和 Converge AI，但必须服务于“仿真顾客帮助企业做决策”这一主线。
